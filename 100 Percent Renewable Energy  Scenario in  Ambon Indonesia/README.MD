# 100% Renewable Energy Scenario for Ambon, Indonesia

This project models a **100% renewable electricity system for Ambon, Indonesia** using the **PyPSA** energy system modeling framework.

The optimization is implemented in **Python** with **PyPSA + Linopy** and solved using the **HiGHS** optimization solver to determine the least-cost renewable generation and storage mix required to meet electricity demand.

**Technologies considered**

* Solar PV
* Wind power
* Battery energy storage
* Grid balancing / system flexibility

**Tools**

* Python
* PyPSA
* Linopy
* HiGHS Solver

The model performs **hourly dispatch optimization** to evaluate system feasibility, costs, and renewable integration for a fully decarbonized power supply in Ambon.
